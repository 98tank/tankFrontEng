### 1: Install NODEJS
[NodeJs](https://nodejs.org/en/) | 
|---|

### 2: Install NPM
[NPM](https://www.npmjs.com/get-npm) | 
|---|
~~~
npm install npm@latest -g
~~~

### 3: Install IONIC CLI
[Ionic CLI](https://ionicframework.com/getting-started) | 
|---|
~~~

~~~

### 4: Install NODE_MODULEÂ´S
